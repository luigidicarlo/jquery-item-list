# Ejemplo de Item List con jQuery y fetch API

## Instalación

No se requiere de instalación para usar el código.

## Ejecución

Se debe emplear algún servidor (Apache, NGINX, Express, etc.) para poder utilizar el código de manera apropiada, pues se hacen peticiones HTTP para obtener la data.

Este es un código de ejemplo. Lo ideal es utilizarlo en un proyecto donde se tenga un setup con base de datos y una REST API de donde se saquen los datos.
